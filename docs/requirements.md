 <h1 style = "text-align: center; border-bottom:none;">User Stories</h1>

1. As a PhD student, I want to be able to navigate between data collection, analysis and venues so that I can quickly find the information I need.

2. As a PhD student, I want to be able to view a list of data sources including names, data types and descriptions so that I can identify which platform best fits my research questions.

3. As a Ph student, I want to be able to compare different collection methods along with their pros and cons so that I can choose the most efficient approach for my data gathering.

4. As a PhD student, I want to be able to browse analysis methods by type and common use cases so that I can determine which learning techniques best fit my data.

5. As a PhD student, I want to be able to filter venues by their type so that I can decide which venue is best to submit my work.
